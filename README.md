# 묵찌빠게임 설명 
---
컴퓨터가 제시하는 임의의 숫자와 사용자가 입력하는 숫자를 비교하여 가위바위보 게임을 진행.

## 페어 프로그래밍 규칙
- 순서도는 먼저 그리고 시작한다.
- 서로 번갈아 가며 하나의 함수를 구현한다.

## Step 1
- 실행 예
> 가위(1), 바위(2), 보(3)! <종료: 02 : 1 비겼습니다!

가위(1), 바위(2), 보(3)! <종료 : 02 : 6 잘못된 입력입니다. 다시 시도해주세요.

가위(1), 바위(2), 보(3)! <종료 : 02 : 3 이겼습니다!
게임 종료

> 가위(1), 바위(2), 보(3)! <종료 : 0> : 0
게임 종료
